The simplified clone of the Unix bc tool.

## Installation

```
$ go get github.com/irenicaa/go-calculator/...
```

## Usage

```
$ go-calculator -h | -help | --help
```

Stdin: code (see [docs](docs/) for details).

Options:

- `-h`, `-help`, `--help` &mdash; show the help message and exit.
